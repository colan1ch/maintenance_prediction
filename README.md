# Predictive Maintenance Dataset

Скрипт ml_solution.py решает задачу предсказания отказа оборудования на основе датасета AI4I 2020. Он загружает данные, делает разведочный анализ и предобработку, обучает две модели (логистическую регрессию и градиентный бустинг) и выводит метрики для порога 0.05


## Установка зависимостей
```bash
pip install pandas, scikit-learn
```

## Запуск
```bash
python ml_solution.py
```

## Этапы решения задачи
1) #### Подключение библиотек и загрузка данных
Читает CSV из `/Users/nachernev/Downloads/ai4i2020.csv`

2) #### EDA
Показывает размер датасета, типы колонок, пропуски, распределение целевой переменной, статистики числовых признаков

3) #### Предобработка
  - Применил Label-Encoding для столбца "Type" (L=0, M=1, H=2)
  - Разделил train/val 80/20 со стратификацией по целевой переменной
  - Применил StandardScaler к данным 

4) #### Модели
Логистическая регрессия и градиентный бустинг

5) #### Метрики
Accuracy, ROC-AUC, classification report с порогом 0.05
